# README

## Getting started

Ensure that you are using the correct Ruby version

```sh
$ asdf current ruby
> 3.3.3
```

Ensure that all the correct dependencies have been installed

```sh
$ bundle install
> Bundle complete!
```

Run the dev environment

```sh
$ ./bin/dev
```
